# 005｜水天需｜等待｜卦码卡视觉优化说明文档 V1

## 一、文档定位

本文件用于锁定《观爻 / GUANYAO SANDBOX》卦码卡 005｜水天需｜《等待》的视觉、文案、版式与工程入仓标准。

005 是 64 张卦码卡中“边界等待型 / 水面阻隔型 / 时机未开型”的基准卡。

005 不得沿用 001｜乾为天｜高空 的中轴、光柱、平台、祭坛、启动装置结构。
005 不得沿用 002｜坤为地｜大地 的厚重地层承载结构。
005 不得沿用 003｜水雷屯｜破土 的裂缝、破开、地层撕裂结构。
005 不得沿用 004｜山水蒙｜迷雾 的雾层遮蔽结构。

005 的核心视觉应从“冲破”转向“等待”，从“立刻行动”转向“时机尚未完全打开”，从“看不清”转向“看得见边界，但还不能贸然越过”。

---

## 二、005 核心定义

编号：005
卦名：水天需
卡名：《等待》
上卦：坎
下卦：乾
母题：等待、时机、阻隔、准备、信任、边界未开、蓄势不发

005 的核心命题：

不是所有等待，
都是拖延。

水天需不是被动停住，也不是逃避行动。
它要指出的是：

有些时机还没有完全打开。
真正的等待不是原地不动，而是在边界前保持准备。

005 要表达的不是“浪漫等待”，不是“海边风景”，也不是“月光水面”，而是：

一个清晰的边界横在前方，力量已经准备好，但还不能贸然跨过去。

## 六、005 主视觉底图规则

### 6.1 主视觉底图定义

005 主视觉底图必须是：

无字、无 LOGO、无编号、无卦名、无卡名、无金句、无标签、无边框、无卦符的纯视觉底图。

所有文字、编号、边框、LOGO、标签、卦名、卡名都由固定模板后期叠加。

### 6.2 005 主视觉必须表达

005 主视觉要表达：

1. 等待
2. 边界
3. 水面阻隔
4. 时机未开
5. 准备但不贸然行动
6. 前方可见但不可轻易越过
7. 安静蓄势
8. 不确定中的稳定准备

005 不是“漂亮水面”，而是“边界前的准备”。

### 6.3 005 可用视觉结构

可用结构方向：

1. 深黑空间中的水平水面边界
2. 水面像一道不可贸然跨越的界线
3. 远处有微弱冷金光，但没有完全打开
4. 下方或近处有稳定结构，表示准备
5. 画面可以有水平分割，但不能变成风景
6. 光源应低位、远处、克制
7. 空间应安静、压住、等待，而非爆发

### 6.4 005 禁止视觉方向

禁止画成：

1. 海景
2. 湖面风景
3. 月亮倒影
4. 日出海平线
5. 浪漫等待
6. 人物站在水边
7. 船
8. 桥
9. 港口
10. 仙境水面
11. 001 式光柱
12. 002 式地层
13. 003 式裂缝
14. 004 式大雾
15. 祭坛
16. 平台
17. 能量反应堆
18. 满屏粒子
19. 文字 / LOGO / 卦符 / 边框

特别禁止：

不要把 005 画成“海边风景”。
不要把 005 画成“等日出”。
不要把 005 画成“水面上的 001”。
不要让光从中心冲天而起。

## 七、005 视觉优化方向

005 应保留：

1. 深黑背景
2. 冷金 / 暗金微光
3. 抽象结构
4. 大面积负空间
5. 水面边界
6. 水平阻隔
7. 克制光源
8. 安静蓄势感

需要强化：

1. 等待不是停摆，而是准备
2. 水面不是风景，而是边界
3. 光不是答案，而是尚未打开的时机
4. 画面不能太亮
5. 不能有强中心光柱
6. 不能有明显风景叙事
7. 要让用户感觉“现在还不能贸然跨过去”

## 合格主视觉判断标准

005 主视觉合格，需要满足：

1. 第一眼看到的是“边界等待”，不是“水景”。
2. 第一眼看到的是“时机未开”，不是“答案出现”。
3. 第一眼看到的是“准备停留”，不是“完全停滞”。
4. 光源克制，不冲天。
5. 主体不祭坛化，不平台化。
6. 没有文字、LOGO、卦符、边框。
7. 能与 001、002、003、004、010 放在同一套卡牌体系中，但骨相完全不同。
8. 适合被固定 A 面模板叠加文字。
9. 用户主观认可。

只要用户不满意，就不得入仓。

## 十二、当前结论

005｜水天需｜《等待》必须作为“边界等待型 / 水面阻隔型 / 时机未开型”的基准卡完成。

005 的任务不是延续 001，也不是延续 003，而是建立“等待与时机”的卦码卡骨相。

001 代表上升型 / 高位承压型。
002 代表承载型 / 地层边界型。
003 代表裂缝破土型。
004 代表迷雾遮蔽型。
005 代表边界等待型 / 水面阻隔型。

005 不得套用 001 的中轴、光柱、平台、祭坛结构。
005 不得套用 003 的裂缝、破开、地层撕裂结构。
005 正式底图必须无字、无 LOGO、无卦符。
005 A 面不得出现卦符。
005 B 面才允许出现固定完整六爻卦形。
