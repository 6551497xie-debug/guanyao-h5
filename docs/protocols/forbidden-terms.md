# 禁用词协议

## 禁止前台出现

- 母码
- MotherCode
- 算命结果
- 命运判词
- 注定
- 必然
- 唯一破局
- 改命
- 逆天改命
- 不付费就会失败
- 你就是某某人格
- 天生王者
- 帝王人格
- 逃避型人格羞辱
- 讨好型人格诊断
- 抑郁诊断
- 病理化人格

## legacy 兼容

允许作为代码 legacy 字段但不得前台外露：

- `motherCode`
- `MotherCode`

## 使用规则

- 禁用词扫描优先面向前台文案、页面文本、展示数据。
- 代码 legacy 字段不作为前台命名依据。
- 若不得不保留 legacy 字段，必须在前台映射为“卦场 / GuaField”。
