# Scene R2C 现实种子协议

该协议属于后续 R2C，不进入当前 R2A。

## 核心公式

元码 × 人生阶段 × 人格切片 × 现实压力层 × 三个细场景行为种子

## 规则

- Identity 锁内因。
- 现实压力层锁范围。
- 三条现实种子锁具体颗粒度。
- 不是泛场景推荐。
- 不是家庭 / 团队 / 伴侣三大类横跳。
- 不是让用户重新选择人格。

## 前台交互

1. 现实信号流滚动。
2. 一组 3 条现实种子滚动呈现。
3. 点击“拦截——正在发生”。
4. 当前 3 条被冻结。
5. 用户点中真正正在发生的一条。
6. 展开现实现场 / 行为惯性 / 重力钩子。
7. 进入观爻卦场｜64 显影。

## 当前 R2A 边界

- 当前 R2A 保持已锁定主链路稳定。
- 不在 R2A 中实现 R2C 动态现实种子流。
- 不恢复旧的“还不是我 / 继续寻找 / 换一幕”判断感。
