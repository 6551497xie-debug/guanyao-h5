# 观爻语言风格协议

观爻语言 = 人格宗仪器语。

## 核心风格

冷白仪器语 + 人格源代码 + 八卦卦解 + 不定罪的残酷剖面。

## 必须保留

- 系统视角
- 短句压印
- 行为重力
- 失重母题
- “你把 X 误认成 Y”句式
- “沙盒不审判……它只记录……”句式

## 禁止写成

- 心理测试
- 性格标签
- 命运判词
- 玄学大师口吻
- 成功学鸡汤
- AI 普通报告
- 情绪安慰文本

## 写作边界

- 不安慰，不定罪。
- 不把用户说死。
- 不把八卦写成玄学判词。
- 不把结果写成最终命运。
- 保持冷、准、短、有行为因果。

## 前台文案必须说人话

观爻可以冷、锋利、有仪器感，但必须让普通用户一眼读懂。

目标用户存在文化差异、认知差异、年龄差异、社会经验差异。前台文案不能依赖咨询黑话、组织黑话、AI 黑话、心理学黑话或玄学黑话。

原则：

- 锋利但不晦涩。
- 冷白但不黑箱。
- 有风格但不装腔。
- 用户要先被刺中，而不是先费力理解术语。

执行规则：

- 按钮要像动作，不像概念。
- 引导语要告诉用户现在发生了什么。
- 状态提示要告诉用户下一步能做什么。
- 保留“元码 / 卦场 / 爻码 / 时间沙漏”等核心 IP 词，但周边解释必须说人话。

## 反例与改写示例

不推荐：

- 替系统旧账继续托底

推荐：

- 明明不是你造成的，最后还是你来扛。
- 别人留下的烂摊子，你一直在收拾。
- 很多人把责任放下了，只有你还在原地托着。
- 问题不是你制造的，但你已经习惯替它收尾。

不推荐：

- 结构性托底
- 长期代偿机制
- 认知闭环
- 底层叙事
- 情绪投射
- 组织惯性

推荐：

- 你一直在替别人收拾残局。
- 你不是不会拒绝，是太怕一拒绝关系就断。
- 你不是没有脾气，是太习惯先把自己压下去。
- 你不是不累，是太久没人问你还能不能扛。

## 适用范围

该规则适用于所有前台用户可见文字，包括：

- 内容母本
- Identity 人格映照碎片
- Scene 现实种子
- 卦场显影
- 爻码卡
- 90天行为防御本
- Archive 回看
- 页面引导语
- 按钮文案
- 状态提示
- 保存提示
- 解锁提示
- 过渡语
