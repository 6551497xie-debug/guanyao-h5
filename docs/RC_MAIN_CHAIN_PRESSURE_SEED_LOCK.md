# RC 主链路锁定｜压力种子捕获层

## 一、链路定义

本文件锁定观爻 1.0 RC 阶段的主体验链路。

压力种子捕获层必须位于：

母码资产生成 / 星兽初显之后，六维空间之前。

压力种子不是独立页面，不是旧卦码页入口，不是额外测试题。

压力种子是母码初显之后，现实引力扰动进入人格行为系统的唯一入口。

---

## 二、锁定链路

```text
[ INPUT: 用户输入时空坐标 ]
         │
         ▼
① Cosmic Field（宇宙粒子场）
   沉浸式暗黑星空启动，星云流体与引力波纹扩散。
         │
         ▼
② 7-Star Core（母码资产生成）
   纵21 / 横7 轴校准，7个金色光点聚火，凝结出静态本命母码。
         │
         ▼
③ Star Beast Emergence（主星兽初显）
   7个光点向外吐纳星尘，抽象连线，四象光兽轮廓在星河中隐现。
         │
         ▼
⚡ Gravity Blackhole（引力黑洞 / 压力种子捕获）
   星河突然暗淡，抛出高共鸣的现实压力种子。
   这颗种子化作一个引力黑洞，打破星兽的初始稳态。
         │
         ▼
④ 6D Cosmic Variables（意识场对齐）
   用户带着压力种子进入六维空间。
   六维行为不是普通打卡，而是提取自我原力，填补和化解引力黑洞。
         │
         ▼
⑤ Supernova Collapse（资产结晶）
   六维能量注入完毕后，引力黑洞被化解。
   星兽破茧而出，最终凝结为一张具有独特时空切片的 64 卦动态人格资产卡。
```

---

## 三、工程约束

1. `LaunchLab` 新链路中，压力种子捕获完成后必须进入六维空间。
2. 新链路不得回落到旧 `/hexagram-stamp` 作为默认中转。
3. 压力种子上下文必须在进入六维空间前完成写入。
4. 六维空间必须读取已捕获的压力种子上下文。
5. 64 卦人格资产卡只能在六维能量注入完成后进入结晶阶段。

---

## 四、禁止行为

禁止：

1. MotherCard 后直接进入旧卦码页。
2. 压力种子捕获绕过六维空间。
3. 六维空间没有压力种子上下文就进入主流程。
4. 压力种子反向改写母码快照。
5. 压力种子反向触发 deterministic persona engine。
6. 将压力种子表现为系统审判、测试题或结果页。

---

## 五、语义约束

母码卡表达的是：

你天生的星河坐标与原力形状。

压力种子表达的是：

现实此刻对这个原力形状产生的引力扰动。

六维空间表达的是：

用户带着星兽与压力种子一起进入行为场，通过身体、情绪、思维、行为、记忆、目标六个维度，为星兽注入化解压力黑洞的能量。

64 卦人格资产卡表达的是：

本局时空切片中，星兽经历压力扰动与六维调频后结晶出的动态人格资产。
