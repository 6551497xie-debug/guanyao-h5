## 1. 语义层定位

本刀属于 Genesis 第七层「兽｜个人星兽归来」，是第一轮视觉语义链的最终显化层。它只把前六层视觉状态融合为显化状态，不改写生命身份或任何计算结果。

```text
月 → 星 → 时 → 象 → 卦 → 力 → 兽
```

输入来自前六层视觉状态，输出为 Personal Star Beast Reveal Visual State。视觉只显化语义，不创造生命。

## 2. 个人星兽定义

星兽不是生成。星兽是前六层生命结构共同归来的显化。

它不是动物、宠物或角色，而是星辰生命体：

```text
星骨 + 象场 + 卦纹 + 原力节律 = 个人星兽
```

## 3. 六层归来融合

### 星辰生命聚合

继承星层的星河秩序、星尘聚合与星骨连接趋势；不直接出现动物，不加载模型，不生成角色。

### 象形承载

继承象层的空间姿态、运动方向与结构气质。四象是形态场，不是青龙模型、白虎模型、朱雀模型或玄武模型。

### 卦纹印记

继承卦层的结构纹理、阴阳节律与变化痕迹；不展示卦卡、卦名或命理解读。

### 原力呼吸

继承力层的呼吸节奏、光流方式与运动倾向；不表现技能释放、能量攻击或属性展示。

月层提供太阴入口，时层提供生命时序进入关系；二者都不被兽层重新计算。

## 4. Reveal 节奏

```text
存在 → 成形 → 归来
```

- 存在：这里有一个生命。
- 成形：它来自刚才所有变化。
- 归来：这是属于我的生命形态。

显化必须连续、安静、有来源感。禁止突然出现、召唤动画或英雄登场。
