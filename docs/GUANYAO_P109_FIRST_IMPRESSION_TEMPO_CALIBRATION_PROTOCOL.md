# GUANYAO P109 First Impression Tempo Calibration Protocol

协议编号：`RC-GUANYAO-GENESIS-FIRST-IMPRESSION-TEMPO-CALIBRATION-P109`

阶段：`GUANYAO Genesis WebGL Prototype Calibration Phase`

状态：`ISOLATED_PROTOTYPE_ONLY / HUMAN_REVIEW_REQUIRED`

## 01｜本刀目标

P109 只校准首次显化的宏观节奏，回应 P100 人工验收中的“有点慢”反馈：

> 加快生命抵达，保留生命在场。

本刀不改变生命身份、显化语法、Scene Model、RenderPlan、Renderer 或正式产品流程。

## 02｜冻结时序

```text
ARRIVAL    0ms → 1400ms
FORMATION  1400ms → 3800ms
PRESENCE   3800ms → 持续停留
```

Fallback 与 WebGL Harness 使用同一组宏观阶段门槛，避免降级路径比主路径更慢或更快。

外层视觉过渡同步收短：

- Canvas：`1200ms / 1400ms / 1600ms`
- Life Halo：`1400ms / 1800ms`
- 控制项显现：`320ms` 延迟、`520ms` 过渡

这些数值只控制到达与显现的外层节拍，不改变星核呼吸、星尘流动或形态场的慢节奏。

## 03｜体验边界

允许：

- 首次可见生命结构更早出现；
- 由宇宙场进入形成阶段的等待缩短；
- Presence 阶段继续安静停留。

禁止：

- 通过加粒子或加亮度弥补节奏；
- 改变 Identity、Manifestation Grammar 或 RenderPlan；
- 改写 Renderer 的生命组织关系；
- 接入 Production、Launch、Gravity、Dynamics、Crystal 或 Storage。

## 04｜人工验收

P109 仍需人工确认：

1. 第一眼是否更快进入“有生命正在靠近”；
2. Formation 是否仍然可被感知，而不是瞬间跳转；
3. Presence 是否仍有安静、古老、在场的停留感；
4. Case A / Case B 的差异是否保持。

工程 Gate 只能确认节奏契约与隔离边界，不能替代第一眼视觉判断。
