# 二、001 核心定义

## 2.1 基础信息

编号：001
卦名：乾为天
卡名：《高空》
上卦：乾
下卦：乾
母题：高位、方向、硬撑、分力、开局、孤立承压

## 2.2 001 的核心命题

001 不是“天光降临”，不是“能量启动”，不是“祭坛开机”，也不是“平台被光柱击中”。

001 要表达的是：

一个人站在高位，承担方向、责任与开局压力，但力量尚未被分出去。

它的核心不是“强大”，而是：

越站在高处，越不能只靠一个人硬撑。

## 2.3 001 的行为含义

乾为天不是赞美用户“强”，而是指出一种高位惯性：

你习惯定方向，习惯先扛起来，习惯让事情从你这里开始。
但当所有力量都集中在你一个点上，开局就会变成硬撑。

001 要提醒用户：

真正的开局，不是一个人顶住所有人，
而是让力量开始流动。

# 四、A 面结构规则

## 4.1 A 面必须出现的内容

A 面是传播面、收藏面、第一眼识别面。

A 面允许出现：

1. NO.001
2. 卦名：乾为天
3. 卡名：《高空》
4. 主视觉
5. 金句
6. 三标签
7. GUANYAO SANDBOX / 观爻

## 4.2 A 面绝对禁止出现的内容

A 面禁止出现：

1. 六爻卦符
2. 上卦 / 下卦说明
3. 卡魂长文
4. 复盘问题
5. 任何系统读数
6. 任何交互提示
7. 任何 UI 风格按钮
8. 任何多余装饰性图标
9. 任何解释性段落

特别锁定：

六爻卦符只允许出现在 B 面顶部信息栏。
A 面不得出现卦符。

## 4.3 A 面信息层级

A 面层级必须是：

第一层：卦名
第二层：卡名
第三层：主视觉
第四层：金句
第五层：三标签
第六层：品牌落款

卦名为主，卡名为辅。
不得把《高空》放大到超过“乾为天”。
不得让视觉装饰压过卦名。
不得让 LOGO 抢主视觉。

# 五、B 面结构规则

## 5.1 B 面定位

B 面是复盘面、收藏说明面、数字翻面与实体卡背面。

B 面不是资产深读页。
B 面不是长报告。
B 面不承载完整卦码资产解读。

## 5.2 B 面允许出现的内容

B 面允许出现：

1. 编号
2. 固定完整六爻卦形
3. 卦名
4. 卡名
5. 上卦 / 下卦
6. 卡魂
7. 复盘问题
8. 品牌落款

## 5.3 B 面排版方向

B 面整体应保持浅米金实体卡背气质。

视觉要求：

1. 米白 / 浅米金底色
2. 金棕色细线
3. 留白充足
4. 分隔线克制
5. 字体清晰
6. 不像 UI 页面
7. 不像说明书
8. 不像报告页

B 面顶部卦形必须是固定六爻卦形库调用，不得由生图模型生成，不得随机绘制，不得错爻。

# 六、A 面主视觉底图规则

## 6.1 主视觉底图与成卡的区别

主视觉底图不是完整卡面。

主视觉底图必须是：

无字、无 LOGO、无编号、无卦名、无卡名、无金句、无标签、无边框、无卦符的纯视觉底图。

所有文字、编号、边框、LOGO、标签、卦名、卡名都由固定模板后期叠加。

## 6.2 001 主视觉必须表达

001 主视觉要表达：

1. 高位
2. 孤立
3. 承压
4. 方向
5. 向上
6. 力量集中
7. 支撑不足
8. 分力尚未发生

它不是“光很好看”，而是“一个高位结构正在承受集中的开局压力”。

## 6.3 001 主视觉可用结构

可用结构方向：

1. 高空中的孤立结构
2. 深黑空间中的承压核心
3. 向上的力量轴
4. 未完全连接的支撑线
5. 高处平台但不可祭坛化
6. 黑场中一个被迫承担方向的结构中心
7. 少量冷金光源从结构缝隙或中心压力点出现

## 6.4 001 主视觉禁止方向

禁止画成：

1. 科幻祭坛
2. 能量反应堆
3. 启动平台
4. 神秘仪式台
5. 天光降临宗教感
6. 高塔风景照
7. 城市建筑海报
8. 星空风景
9. 仙侠云海
10. 满屏金线网络
11. 满屏粒子光点
12. 机械装置
13. UI 界面
14. 人物站在高处
15. 卦符悬浮在画面里

特别禁止：

不要再出现“中心光柱打在平台上”的装置感。
不要再把 001 画成“平台被能量击中”。
不要再把 001 画成“祭坛启动”。

# 七、001 视觉优化方向

## 7.1 保留方向

如果使用既有乾母版作为基础，应保留：

1. 深黑高空感
2. 竖向上升张力
3. 中心结构感
4. 冷金光源
5. 高处孤立感
6. 大面积黑场
7. 高位承压气质

## 7.2 必须优化的问题

需要优化：

1. 去掉所有文字
2. 去掉卦符
3. 去掉 LOGO
4. 去掉边框
5. 去掉卡面排版
6. 减少装饰性金线
7. 减少粒子光点
8. 压弱宗教 / 祭坛 / 启动装置感
9. 强化“高位责任”
10. 强化“支撑未连接”
11. 强化“力量尚未分出去”
12. 保持画面克制，不要炫技

## 7.3 合格主视觉判断标准

001 主视觉合格，需要满足：

1. 第一眼看到的是“高位承压”，不是“光柱好看”。
2. 第一眼看到的是“孤立硬撑”，不是“祭坛启动”。
3. 画面主体明确，不散。
4. 黑场足够，不拥挤。
5. 光源克制，不爆。
6. 没有文字、LOGO、卦符、边框。
7. 能与 003、010 放在同一套卡牌体系中。
8. 适合被固定 A 面模板叠加文字。
9. 用户主观认可。

只要用户不满意，就不得入仓。

# 十一、当前结论

001｜乾为天｜《高空》必须先作为黄金母版完成。

在 001 未通过前，不继续批量推进 002—064。

001 通过后，后续 64 张卦码卡统一采用以下流程：

原始母版 → 优化候选 → 用户确认 → 正式入仓 → 模板调用 → build 验收

不得再走：

Codex 生图 → 失败候选 → 误入参考 → 反复清理

最终锁定：

图片审美与优化由用户和 ChatGPT 完成。
工程入仓与路径调用由 Codex 完成。
A/B 面排版由固定模板完成。
卦符只出现在 B 面。
A 面永不出现卦符。
正式底图永远无字、无 LOGO、无卦符。
