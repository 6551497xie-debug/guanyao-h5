# GUANYAO Star Mansion Life Trajectory Source Freeze Protocol

协议编号：`RC-STAR-MANSION-LIFE-TRAJECTORY-SOURCE-FREEZE-P89`

## 1. 本刀定位

P89 冻结 GUANYAO 星兽显化的最高体验语义源。它只建立来源引用、职责边界与体验顺序，不创建新的产品架构、生命实体、Journey Runtime、Renderer、视觉资产或持久化结构。

视觉不能决定生命身份。视觉只能显化正式引擎已经确定的生命身份。

## 2. 生命轨迹

最高体验链固定为完整十二阶段：

```text
COSMIC_SKY
→ TWENTY_EIGHT_MANSION_REVEAL
→ LIFE_ARRIVAL_COORDINATE
→ TIME_SEQUENCE_ALIGNMENT
→ BIRTH_MANSION_IGNITION
→ FOUR_SYMBOL_ALIGNMENT
→ MOTHER_CODE_INFUSION
→ PERSONAL_STAR_BEAST_REVEAL
→ REALITY_PRESSURE
→ GRAVITY_SIX_SPACE_EXPERIENCE
→ CHOICE_PERSONA_MIGRATION
→ CRYSTAL_IMPRINT
```

对应的产品体验语义是：

```text
宇宙星空
→ 二十八宿星兽显影
→ 生命降临坐标输入
→ 时序定位
→ 本命星宿点亮
→ 所属四象归位
→ 母码原力落入
→ 个人星宿兽显化
→ 现实压力
→ Gravity / 六维生命体验
→ Choice / 人格迁移
→ Crystal / 生命印记
```

该顺序是体验语义，不是新的计算依赖，也不授权本刀推进 Journey Stage、制造压力、运行 Gravity、执行人格迁移或创建 Crystal。

## 3. 正式来源

`StarMansionLifeTrajectorySource` 只保存五项正式引用：

1. 生命降临坐标引用；
2. 本命星宿正式结果引用；
3. 四象正式结果引用；
4. `MotherCodeProfile` 正式结果引用；
5. 由该 `MotherCodeProfile` 映射得到的 `LifeArchetypeProfile` 引用。

其中星宿与四象来自既有 Star Beast Engine 的同一次正式结果，Mother Code 与 Life Archetype 保持既有正式来源桥接。本刀不重新计算任何结果，也不复制结果事实。

## 4. 三层职责边界

### 本命星宿：生命种子

本命星宿回答：“这个生命从哪颗星辰结构开始显化。”

- 四象不得替代本命星宿；
- Mother Code 不得替代或改变本命星宿；
- 本命星宿不得反推生命原力。

### 四象：生命形态场

四象回答：“这个星辰生命以何种宇宙形态归象。”

- 四象只提供 `FOUR_SYMBOL_MORPHOLOGICAL_FIELD`；
- 四象不是个人星宿兽本身；
- 四象不得直接生成动物模型；
- 四象不得反推 Mother Code 或 Life Archetype。

### Mother Code / Life Archetype：生命原力

Mother Code 回答：“这个生命携带什么力量来到世界。”

正式来源保持：

```text
MotherCodeProfile
→ LifeArchetypeProfile
→ LIFE_ARCHETYPE_FORCE
```

- Mother Code 不生成四象；
- Life Archetype 不是卦象，也不是人格标签；
- 生命原力不得改变本命星宿来源。

## 5. Personal Star Beast 汇合定义

个人星宿兽身份来源固定为：

```text
MansionSeed
+ FourSymbolField
+ LifeArchetypeForce
→ PersonalStarBeastIdentityReference
```

三项来源在汇合前保持独立，不得互相推导。`PersonalStarBeastIdentityReference` 只证明正式来源已经汇合；它不是四象动物、不是新 Life State、不是已生成的视觉资产，也不在本刀创建 Personal Star Beast 实体。

## 6. 冻结边界

P89 不修改：

- 二十八宿、四象与 Mother Code 既有引擎；
- Original Self、StarBeastState 与 Life State；
- Gravity、六维 ChangeExperience、Choice、Persona Migration 与 Crystal；
- P84–P87 资产、Renderer、Canvas 与隔离预览；
- UI、Storage 与正式 Runtime。

后续 `Star Mansion Beast Manifestation` 施工必须先消费 P89 的正式来源引用，不得从四象动物实验资产反向定义个人星宿兽身份。
