# GUANYAO White Tiger Genesis Asset Protocol

协议编号：`RC-STAR-BEAST-GENESIS-PROTOTYPE-ASSET-P84`

本协议建立第一份 `StarBeastGenesisPrototypeAsset`：艮之白虎 Genesis Prototype。它是未来 Genesis Renderer Prototype 的输入定义，不是图片、模型、动画或视觉成品。

本协议与既有 `RC-STAR-BEAST-PROTOTYPE-RENDERER-EXECUTION-READINESS-P84` 属于不同协议流；尾号相同不代表替代或覆盖，旧 P84 的 Renderer 治理边界保持不变。

## 1. 来源边界

两条来源独立成立，只在资产定义层汇合：

```text
西方七宿（奎、娄、胃、昴、毕、觜、参）
→ 白虎四象身份
→ 艮之白虎资产定义

MotherCodeProfile
→ LifeArchetypeProfile / GEN / 艮
→ 艮之白虎资产定义
```

西方七宿与白虎只提供星兽身份来源；`GEN` 的唯一正式来源仍是 `MotherCodeProfile → LifeArchetypeProfile`。禁止以白虎、四象方位或星宿反推艮，禁止经过 fourSymbol 生成生命原型。

## 2. 西方七宿星骨

西方七宿星骨不是动物轮廓、动物线稿或白虎图片。它以七个星辰节点形成生命结构，视觉关键词为：古老、稳定、秩序。

## 3. 白虎形态

白虎不是战斗虎或凶兽，不以攻击、毛发或写实动物塑形。它是宇宙守望生命，表达安静、威严、守护与在场。

## 4. 艮之原力

艮在此定义为守望者。它不是停止或僵化。视觉表达只保留山岳稳定、边界光与沉稳内核三项资产语义，不包含渲染参数。

## 5. 五层表达资产

1. `MANSION_BONE`：西方七宿星骨。
2. `FOUR_SYMBOL_FORM`：白虎宇宙守望形态。
3. `LIFE_FORCE_CORE`：艮之守望原力与稳定星核。
4. `COSMIC_CONSCIOUSNESS`：星尘、呼吸与意识的表达入口。
5. `CRYSTAL_READINESS`：未来生命印记入口，不创建 Crystal。

## 6. Genesis 五阶段

1. `COSMIC_FIELD`：深空，等待显化。
2. `WESTERN_MANSION_ALIGNMENT`：西方七宿星点出现。
3. `WHITE_TIGER_FORMATION`：七宿形成白虎星骨生命结构。
4. `GEN_INFUSION`：艮之原力进入，星核趋于稳定。
5. `WHITE_TIGER_REVEAL`：初代艮之白虎星兽显现。

五阶段是资产状态定义，不执行动画，不产生像素，不调用 Renderer。

## 7. 工程边界

本协议禁止图片生成、3D 模型、写实动物、毛发、WebGL、Three.js、Renderer、Canvas、UI、Runtime、Storage 与既有主链修改。资产定义不得被 Launch、Gravity、Dynamics、StarbeastLab 或 P88 预览消费。
