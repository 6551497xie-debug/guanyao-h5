# 002｜坤为地｜大地｜卦码卡视觉优化说明文档 V1

## 一、文档定位

本文件用于锁定《观爻 / GUANYAO SANDBOX》卦码卡 002｜坤为地｜《大地》的视觉、文案、版式与工程入仓标准。

002 是 64 张卦码卡中“承载 / 地层 / 边界承受型”的基准卡。

002 不得沿用 001｜乾为天｜高空 的中轴、光柱、平台、祭坛、启动装置结构。
001 只代表“上升型 / 高位承压型”，不能作为 002 的视觉母版。

002 的核心视觉应从“高空向上”转为“地层向下”，从“单点硬撑”转为“大面积承载”，从“开局定方向”转为“边界承受与吞下压力”。

---

## 二、002 核心定义

编号：002
卦名：坤为地
卡名：《大地》
上卦：坤
下卦：坤
母题：承载、忍让、吞下、边界、厚重、地层、容纳、过度承受

002 的核心命题：

你能承载很多，不代表什么都要吞下。

坤为地不是赞美用户“能忍”“能包容”“能接住一切”。
它要指出的是：

当承载变成没有边界，稳定就会变成吞咽。
当大地什么都接住，压力也会一层层沉到身体里。

002 要表达的不是“伟大的母性”，不是“温柔大地”，不是“自然风景”，而是：

一片厚重的地层正在承受过多压力，边界已经开始下沉。

* 开局
* 单点承压

## 六、002 主视觉底图规则

### 6.1 主视觉底图定义

002 主视觉底图必须是：

无字、无 LOGO、无编号、无卦名、无卡名、无金句、无标签、无边框、无卦符的纯视觉底图。

所有文字、编号、边框、LOGO、标签、卦名、卡名都由固定模板后期叠加。

### 6.2 002 主视觉必须表达

002 主视觉要表达：

1. 厚重
2. 承载
3. 下沉
4. 地层
5. 被压住
6. 边界承受
7. 忍让累积
8. 吞下过多压力
9. 稳定中出现沉降
10. 大面积结构承压

002 不是“漂亮大地”，而是“承载过量后的边界压力”。

### 6.3 002 可用视觉结构

可用结构方向：

1. 深黑空间中的厚重地层截面
2. 多层沉积结构向下压缩
3. 大面积承载面承受上方压力
4. 地层内部出现微弱冷金裂隙
5. 地面边界出现下沉或塌陷迹象
6. 厚重板块正在接住外部重量
7. 暗金光源从地层深处或裂缝中低位透出
8. 结构水平展开，而不是垂直冲天

## 七、002 视觉优化方向

### 7.1 保留方向

002 应保留：

1. 深黑背景
2. 冷金 / 暗金光源
3. 抽象结构
4. 大面积负空间
5. 厚重横向结构
6. 低位光源
7. 沉积感
8. 稳定但受压的气质

### 7.2 必须强化的问题

需要强化：

1. 大地不是风景，而是承载结构
2. 承载不是温柔，而是边界压力
3. 地层不是装饰，而是压力沉积
4. 光源必须低位、克制、从结构内部透出
5. 画面要有“吞下太多”的沉默重量
6. 主体应横向铺展，而非竖向上冲
7. 画面气质要稳、沉、重、压，而不是亮、升、爆、神圣

### 7.3 合格主视觉判断标准

002 主视觉合格，需要满足：

1. 第一眼看到的是“大面积承载”，不是“中轴启动”。
2. 第一眼看到的是“厚重地层”，不是“自然风景”。
3. 第一眼看到的是“边界受压”，不是“漂亮大地”。
4. 光源不冲天，不成为光柱。
5. 主体不祭坛化，不平台化。
6. 没有文字、LOGO、卦符、边框。
7. 能与 001、003、010 放在同一套卡牌体系中，但骨相完全不同。
8. 适合被固定 A 面模板叠加文字。
9. 用户主观认可。

只要用户不满意，就不得入仓。

001 代表上升型 / 高位承压型。
