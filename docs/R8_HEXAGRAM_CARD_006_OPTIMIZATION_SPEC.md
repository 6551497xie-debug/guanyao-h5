# 006｜天水讼｜争鸣｜卦码卡视觉优化说明文档 V1

## 一、文档定位

本文件用于锁定《观爻 / GUANYAO SANDBOX》卦码卡 006｜天水讼｜《争鸣》的视觉、文案、版式与工程入仓标准。

006 是 64 张卦码卡中“对峙边界型 / 立场撕裂型 / 争鸣张力型”的基准卡。

006 不得沿用 001｜乾为天｜高空 的中轴、光柱、平台、祭坛、启动装置结构。
006 不得沿用 002｜坤为地｜大地 的地层承载结构。
006 不得沿用 003｜水雷屯｜破土 的破土裂缝结构。
006 不得沿用 004｜山水蒙｜迷雾 的雾层遮蔽结构。
006 不得沿用 005｜水天需｜等待 的悬置等待结构。

006 的核心视觉应从“一个点承压”转向“两侧对峙”，从“能量爆发”转向“边界被迫显现”，从“争输赢”转向“立场不能再被吞掉”。

## 二、006 核心定义

编号：006
卦名：天水讼
卡名：《争鸣》
上卦：乾
下卦：坎
母题：争鸣、立场、边界、对峙、撕裂、不可吞掉、无法继续合拢

006 的核心命题：

你不是非要争赢，
只是不能再替沉默让路。

天水讼不是吵架，不是战争，不是法庭审判，也不是攻击对方。
它要指出的是：

一个人被吞掉太久之后，边界终于不得不显现。
争鸣不是为了压过别人，而是让自己的立场被看见。

006 要表达的不是“冲突爆炸”，而是：

两侧结构已经无法继续假装合在一起，中间那条边界终于亮了出来。

## 六、006 主视觉底图规则

### 6.1 主视觉底图定义

006 主视觉底图必须是：

无字、无 LOGO、无编号、无卦名、无卡名、无金句、无标签、无边框、无卦符的纯视觉底图。

所有文字、编号、边框、LOGO、标签、卦名、卡名都由固定模板后期叠加。

### 6.2 006 主视觉必须表达

006 主视觉要表达：

1. 对峙
2. 立场
3. 边界
4. 撕裂
5. 两侧分开
6. 中间不可忽略的缝
7. 不再沉默
8. 不再被吞掉
9. 无法继续假装合在一起
10. 边界终于亮出来

006 不是“灾变爆炸”，而是“立场边界显现”。

### 6.3 006 可用视觉结构

可用结构方向：

1. 左右两侧黑色几何高墙对峙
2. 中间有一条细窄冷金边界裂隙
3. 光从两侧无法合拢的缝里透出
4. 两边结构彼此压近，但没有融合
5. 画面上中部张力强，下方保留暗场给金句和标签
6. 可有少量碎块，但不得变成灾变特效
7. 可以有纵深，但主体必须是左右对峙，不是城市风景

### 6.4 006 禁止视觉方向

禁止画成：

1. 法庭
2. 战场
3. 人物争吵
4. 武器
5. 旗帜
6. 城市大战
7. 火焰爆炸

9. 001 式祭坛平台
10. 002 式地层承载
11. 003 式破土裂缝
12. 004 式迷雾遮蔽
13. 005 式悬置等待
14. 底部圆形反应堆

16. 能量炮
17. 满屏漂浮碎块
18. 满屏粒子
19. 文字 / LOGO / 卦符 / 边框

特别禁止：

不要把 006 画成“光柱把城市劈开”。
不要把 006 画成“祭坛启动”。

## 七、006 视觉优化方向

006 应保留：

1. 深黑背景
2. 冷金 / 暗金边界光
3. 抽象几何结构
4. 左右对峙墙体
5. 中央边界缝
6. 空间纵深
7. 黑金文明切片气质
8. 下方暗场空间

需要强化：

1. 争鸣不是爆炸，而是边界显现
2. 光不是神圣降临，而是从缝里透出
3. 中轴不能像 001
4. 底部不能出现祭坛圆盘
5. 碎块要克制，不做灾变特效
6. 左右对峙必须比中央光线更重要
7. 底部 25% 需要给金句、标签、LOGO 留出呼吸空间

## 合格主视觉判断标准

006 主视觉合格，需要满足：

1. 第一眼看到的是“左右对峙”，不是“中轴光柱”。
2. 第一眼看到的是“边界显现”，不是“能量爆发”。
3. 第一眼看到的是“立场分开”，不是“灾变现场”。
4. 光源克制，从缝里透出。
5. 主体不祭坛化，不平台化。
6. 没有文字、LOGO、卦符、边框。
7. 下方留有足够暗场空间，适合 A 面叠加金句和标签。
8. 能与 001、002、003、004、005、010 放在同一套卡牌体系中，但骨相不同。
9. 用户主观认可。

只要用户不满意，就不得入仓。

## 十二、当前结论

006｜天水讼｜《争鸣》必须作为“对峙边界型 / 立场撕裂型 / 争鸣张力型”的基准卡完成。

006 的任务不是延续 001，也不是延续 003，而是建立“边界显现与立场对峙”的卦码卡骨相。

001 代表上升型 / 高位承压型。
002 代表承载型 / 地层边界型。
003 代表裂缝破土型。
004 代表迷雾遮蔽型。
005 代表边界等待型 / 水面阻隔型。
006 代表对峙边界型 / 立场撕裂型。

006 不得套用 001 的中轴、光柱、平台、祭坛结构。
006 不得套用 003 的裂缝、破开、地层撕裂结构。
006 正式底图必须无字、无 LOGO、无卦符。
006 A 面不得出现卦符。
006 B 面才允许出现固定完整六爻卦形。
