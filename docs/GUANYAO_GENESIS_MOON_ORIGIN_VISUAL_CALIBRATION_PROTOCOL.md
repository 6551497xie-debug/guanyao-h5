# RC-GUANYAO-GENESIS-MOON-ORIGIN-VISUAL-CALIBRATION-P1

# GUANYAO Genesis Moon Origin Visual Calibration V1.0

本刀把 Genesis 第一层「月｜太阴入口」落为视觉状态与映射契约，不改变任何生命计算或正式产品流程。

## 层级范围

```text
月｜太阴入口
```

本刀只处理月层，不提前进入星、时、象、卦、力、兽。

## Moon Origin Visual State

月层表达太阴入口：

- `lunarPresence = TAIYIN_PRESENT`
- `moonPhaseExpression = QUIET_ROUND_MOON`
- `cosmicDepth = UNLOCATED_DEEP_SPACE`
- `moonlightField = SOFT_TAIYIN_FIELD`
- `temporalRhythm = SLOW_STABLE_BREATH`
- `entranceState = ENTERING_TAIYIN_REALM`

它表达的是天地时间场的进入，不是生命身份、星宿结果或星兽结果。

## 第一屏表达规则

深空应有层次、微弱星尘与空间感，但不使用科幻 HUD 或游戏背景语言。

月象应是安静的圆月：具备球体存在感、柔和月华、微弱表面层和安静呼吸；不得表现为能量球、爆发光源或魔法核心。

月华和整体空间节奏保持慢、稳、克制：不闪烁、不快速流动、不使用强粒子效果。目标是让用户感到“进入一个地方”，而不是观看一段动画。

## 第一屏文案

主语义：

> 日月运行，星辰有序。

副语义：

> 你的星位，映照了你来到这个世界时的样子。

文案属于视觉层呈现，不增加解释性说明。

## 严格边界

Moon Visual State 不包含 Identity、Birth Data、Mansion、Four Symbol、MotherCode 或 Star Beast。

映射服务只验证月层语义并生成视觉表达状态，不调用 Engine、Identity Source、Scene Model、RenderPlan 或 Renderer，不污染后续 Genesis 层。

本刀不修改生命计算、四象、星宿、卦象、MotherCode、Personal Star Beast、真实用户或产品 UI。
