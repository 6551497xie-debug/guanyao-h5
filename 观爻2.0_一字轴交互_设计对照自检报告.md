# 观爻 2.0 ·「一字轴 × 界面交互」设计对照自检报告

> 审计性质：只读源码逐屏比对，**未改动任何代码、主链路、内容引擎**。
> 审计基准：①最高创世宪法 ②三大因果连动铁律 ③六阶变奏总装 ④三终极效果。
> 评级图例：✅ 达成 ｜ 🟡 部分达成 ｜ ❌ 缺失 ｜ ⚠️ 与宪法直接冲突（违宪）。

---

## 0. 一句话结论

当前实现已经把「**一根 1px 线贯穿全链**」的**骨架**真实落地了（跨屏粒子守恒、十字调频、裂缝开窗、沙化下坠、三色纪律都在），
但**三大铁律里有两条尚未成立、一条只到半成品**，且存在 **3 处直接违反宪法的「网页性回潮」**。

也就是说：**地基对了，但最能产生「革命性心流」的那三个核心动作（向心收缩 / 雷达硬显 / 反本能拉断）目前要么缺席、要么被网页式淡入淡出稀释了。** 这正是你担心的「做不好会与用户心流严重分歧」的高危区。

---

## 1. 最高宪法对照：空间(黑) → 密度(蓝) → LOGO → 路径(1px线)

| 宪法层 | 当前实现 | 评级 |
|---|---|---|
| 空间（纯黑结构隔离） | 全屏 `#000/#020306` 底，外壳自适应竖屏列，确实是黑色深渊基底 | ✅ |
| 密度（中央 `#00B8D4` 极慢呼吸蓝墨水） | 各屏有冷蓝径向背光场（`createRadialGradient`），但**是静态/缓动，未与任何手势绑定** | 🟡 |
| LOGO（终态结构） | Launch 有完整的「12 段平铺短线 → 断裂升起 → 收敛成 X_MATRIX LOGO」，颜色 `FLOW_BLUE→LOGO_WHITE` 单一真源 | ✅ |
| 路径（1px 线 = 因果传动轴） | `axisHandoff` 让上一屏沙粒在下一屏重新凝结进入场轴线，**跨屏一线贯穿是真的成立的** | ✅（最强资产） |

**结论：宪法的「黑 / LOGO / 线」三层成立，唯独「密度蓝」目前只是背景画，不是会因你拉线而畸变的引力场——这直接牵出铁律 1 的缺失。**

## 2. 三大因果连动铁律 · 逐条审计（最关键章节）

### 铁律 1 ｜ 向心收缩（蓝色星系随进度高压聚集、黑洞畸变） → ❌ 缺失

- **采集证据**：
  - `DefaultReactionScreen.tsx:323` → `createRadialGradient(cx, splitY, 0, cx, splitY, max*0.7)`，半径/圆心**写死**。
  - `MotherFieldEngine.tsx:330` → 同样写死的径向场。
  - 全工程**没有任何一处**把渐变半径 / 圆心 / 浓度绑定到 `m.swipe` / `m.progress`。
- **判定**：用户拉动生命轴时，中央蓝色星系**纹丝不动**。宪法要的「他能用眼睛采集到——因为他拉了轴，黑洞正在畸变」这条最核心的因果反馈**完全没有**。
- **后果**：失去「我在驱动这个时空场」的体感，进度条沦为普通进度条。

### 铁律 2 ｜ 因果雷达：文案随线轨「切片曝光」硬显 → ❌ 缺失，且 ⚠️ 反被违宪手法占位

- **采集证据**：
  - 全工程**搜不到**任何「档位命中 → 文字爆亮」机制（无 sweep / radar / flash / reveal 逻辑）。
  - 文案显隐**清一色用 `smooth()` + `globalAlpha` 做淡入淡出**：`DefaultReaction smooth×8 / globalAlpha×22`，`MotherField globalAlpha×26`，`Chrono smooth×5 / globalAlpha×26`。
- **判定**：宪法白纸黑字「**全场严禁任何 CSS 渐隐(Opacity)**」「未扫描到的文字完全不可见、扫过啪地爆亮」——当前做法**恰好是被明令禁止的那一种**（柔和 alpha 淡入）。
- **后果**：这是**最严重的心流分歧点**。雷达硬显是「冷酷仪器采集感」的灵魂；换成网页式淡入，整套敬畏仪式塌成「又一个会渐变的 H5」。

### 铁律 3 ｜ 物理沙化垂直下坠（拉断瞬间，心魔文字碎成 400 颗自由坠落） → 🟡 半成品

- **采集证据**：
  - 沙化**存在且方向正确**（向下、用真粒子非 opacity）：✅ 这条精神在。
  - 但**粒子量不达标**：Chrono `104+56`、Reaction `150`、MotherField `130`、Hexagram `180+120`——**没有一屏到 400**。
  - **下坠物理两套打架**：
    - `PressureSeed:257` `vy = 2.1 + random*3.8`、`Hexagram:187` `vy = 1.2 + random*4.2` → **逐帧自由落体**，符合宪法 `vy=2.0+random*3.5` 的写法。
    - 但 `Chrono/Reaction/MotherField` 用 `vy = random*70+40` 配 `vy += 320~340*dt` → **另一套 dt 缩放体系**。两套混用 = 各屏坠落手感不一致。
  - **触发时机错位**：当前沙化由「滑到 100% / 点击完成」被动触发；宪法要的是「用户**反本能左滑 / 手撕把线狠狠拉断**的第 0.01 秒」主动触发。
  - **沙化的对象不对**：Reaction 里认知文案是被 `if(!sandifying)` **硬切消失**，真正沙化的是「卡片/CODE/轴」这些代理物——**刺痛用户的心魔文字本身没有碎裂**。
- **判定**：有沙化、方向对、不靠 opacity——但**数量、物理统一性、触发动作、沙化主体**四项都偏离。
- **后果**：「大解脱感」打折——崩断的不是他的心魔，是一张卡片。

## 5. 你最该警惕的 3 个「心流分歧」高危项（违宪级）

> 这三项不是「做得不够好」，是「方向被悄悄改成了它要反对的东西」。优先级最高。

1. **⚠️ 透明渐隐全面占位（违反铁律 2 + 铁律 3 的全局禁令）**
   文案/线/卡几乎全靠 `globalAlpha` 淡入淡出。宪法的「冷酷采集感」整个建立在「硬显/硬碎、绝不渐隐」上。这是当前与心流分歧的**第一震源**。

2. **⚠️ 网页卡片 / 网页清单复活（违反「全屏找不到任何网页盒子/清单」）**
   `DefaultReactionScreen` 的圆角母码卡、`ArchivePage` 的 DOM 清单，正是宪法要「物理注销」的那种网页组件。它们一出现，敬畏仪式就被拉回「普通 H5」。

3. **❌ 向心收缩缺席（铁律 1 整条没落地）**
   拉线时蓝色引力场不畸变，用户拿不到「我在驱动因果黑洞」的核心反馈。革命性的体感有一半就压在这条上。

## 6. 优化建议（按优先级，方向性、不含代码，待你定盘后再动手）

### P0 · 先把「会与心流分歧」的三处违宪扳回来
- **雷达硬显模型**：把文案显隐从 `smooth()+globalAlpha` 渐隐，改成「扫描线（dotX / 线轨位置）经过该文字的 x/档位锚点 → 该段**瞬间硬切到 alpha=1**；未命中 → **根本不绘制**」。即「命中即曝光、错过即死黑」，杜绝中间灰度。
- **向心收缩**：把中央径向场的**半径与浓度绑定到 `m.swipe/m.progress`**（进度越高，半径越收、`#00B8D4` 浓度越聚、可叠加极轻微的坐标向心位移），让「拉线→黑洞畸变」可被肉眼采集。
- **网页盒子去物质化**：母码卡 / Archive 清单从「圆角盒子 + 列表」改为「从 1px 线裂缝/线轨上**长出来的刻字**」，信息依附线轨而非卡片（与 02 已有的「裂缝开窗」语言统一）。

### P1 · 把沙化与拉断升级成「肉身高潮」
- **沙化统一物理**：全链统一到逐帧自由落体口径（如 `vy = 2.0 + random*3.5`），**粒子量向 400 看齐**（按屏幕面积可缩放，但手感档位统一）。
- **触发改为「反本能拉断」**：沙化由「滑满自动」改为**用户主动撕断/反向猛拉**触发；断裂瞬间线头加 `Math.sin` 高频衰减回弹。
- **沙化主体归位**：让**真正刺痛用户的那段心魔文字**直接碎成粒子（而非先硬切消失、再让代理卡片沙化）。
- **递进式触感**：换挡跳格（短脆）、牵引反抗（持续弱震随张力增强）、重力秤（按压时长→震幅指数增）、崩断（单次最高强度暴震）——做成一条触觉强度曲线。

### P2 · 补齐变奏与商业飞轮（结构性，量大）
- 00 补「阴爻 `--` 撕开 → 焊成阳爻 `—` → 12 连杆 45° 折叠成双 X」的语义化创世。
- 03 补「胡克弓弦弯曲」+「准星 180° 3D 翻转 + 更替音」，并考虑前 3 空间分屏。
- 04 补「垂直重力秤指数阻尼 / 左滑像素风化拖尾 / 丝滑吐卡」。
- 05 把 Archive 改为「因果熔断器：右滑充能→7 灯爆亮→熔断→年轮卡」，再接「沙漏 3D 翻转付费墙（9.9 / 99）」。
- 顶部常驻「1px 六爻真理钢印」替换文字眉标，强化仪器感。

## 7. 别推翻的地基（当前真正的资产）

- **`axisHandoff` 跨屏粒子守恒** —「一线贯穿全链」是真的在跑，这是整套宪法最难、也最对的一块，**务必保留**。
- **01 Chrono 十字调频** — 与设计吻合度最高，可作为「线轨即仪器」的样板屏。
- **02 裂缝开窗 The Split** — 裂开/愈合机制已成立，是 P0「去卡片化」可以直接嫁接的语言。
- **三色纪律（灰/蓝/金）+ VOID 四拍入场** — 视觉语法统一，地基稳。
