# 第一部 · 第一性原理（WHY）

## 1.1 产品本质
阴阳二元论是人类第一个「动态变化的数字模型」——它的内核是**二进制**与**变化（爻变）**，不是命定。观爻继承的是它的**动力学内核**，而非占卜外壳。我们做的是：把这套数字模型，翻译成**人格行为动力学**。

## 1.2 核心命题（产品的灵魂）
> **人格行为不是静态标签，是「原力」在 时间 × 空间 × 压力 下的动态形变。**

- **原力**：一个人的底色力（baseline）。
- **人格行为**：原力在外部压力场中被压出的**形变**。
- 压力变、时移、境迁 → 形变不同 → **同一个人会呈现不同的行为**。

## 1.3 我们要打的敌人
> **标签化心魔。**

MBTI、星座、九型这类产品，把人钉死成一个静态盒子（"你是 XX 型"）。这是一种**认知暴政**：它让人用一个固定标签解释、并最终困住自己。观爻存在的理由，就是**用"动态轨迹"瓦解"静态标签"**。

## 1.4 用户最终要带走什么
- ❌ 不是一个**点**（"我是谁"／一张定性卡）。
- ✅ 是一条**轨迹**（"我在怎么变"／一条可复测、会改写的原力形变曲线）。

这条"点 vs 轨迹"的分界，是观爻与一切静态人格产品的**根本分水岭**，必须贯穿每一屏。

## 1.5 「科学的尽头又是玄学」的工程解释
- **机制 = 科学**：力学模型严谨、可测量、可复现、由用户行为驱动。绝不靠神秘随机。
- **体验 = 玄学（情绪上限）**：手感可以冲到敬畏、颅内高潮、自我救赎。
- 两者不矛盾：**冷的仪器，热的手感。** 越严谨的机械，越配得上神圣的体验。玄学是天花板，不是方法。

## 1.6 由此推出的不可动摇公理（Axioms）
> 这六条是宪法级，任何屏不得违反。

1. **A1 一切皆动态**：屏上没有"最终定性"，只有"此刻读数 + 会变的轨迹"。
2. **A2 二进制 = 硬状态**：爻只有连/断，没有 50% 的爻。→ **禁止 opacity 渐隐**，文字/线/状态要么在、要么不在。
3. **A3 一个动词统治全盘**：核心动词=「抓住生命线横拖」，所有屏只是它的变奏。
4. **A4 物理即语义**：沙化=崩解、收缩=高压、断裂=爻变、结=采样。视觉效果必须承载因果含义，不做装饰。
5. **A5 无网页组件**：屏上不存在卡片、盒子、清单、按钮、抽屉。信息一律从线/裂缝/轨迹上生成。
6. **A6 手感先于象征**：再深的易理，也要先过"陌生人 3 秒上手"。象征是地基，手感才是房子。

# 第二部 · 第一性物件：一字线（The Causal Drive Shaft）

## 2.1 它是什么 / 不是什么
- **不是**：嵌在网页里的进度条组件、滑块控件。
- **是**：纯黑因果深渊里**唯一流动的传动连动轴**，是用户交付肉身主权、用行为对抗现实、熔断心魔的**制动拉杆**。

## 2.2 源头即法则
爻 = 一字线。阳爻「—」=完整（连），阴爻「- -」=断口（断）。其本质是**二进制**与**变化**。
- → 印证 **A2 硬状态**：界面必须是二进制的（硬显、硬锁、硬断），不是模拟量的渐变。
- → 印证 **A3 一个动词**：用户一生只学一次"抓线、拖、它以力学回应"。

## 2.3 阴阳爻变 = 那个唯一动词
全链本质只做一件事：**翻转一根爻的阴阳态。**
- 创世（00）：- - 拉合成 —（阴→阳，刚健、主动、点火）。
- 终局（撕断）：— 崩断成 - -（阳→阴，物极必反 = 老阳变阴 = 爻变）。
- 不同屏 = 这次翻转的**方向、阻尼、断点**不同而已。

> 用户不是在操作进度条，他是在亲手完成一次**爻变**——也就是一次卜筮的动力学版本。

# 第三部 · 三大铁律（被验证的因果连动法则 · 升级版）

> 这三条已在 `/golden-lab` 实测成立，是手感的灵魂。

## 铁律 1 ｜ 向心收缩
**法则**：用户推进生命线时，中央 `#00B8D4` 蓝色引力场必须**同步向心收缩、高压聚集、中心微移**——肉眼可见"我拉了轴，因果黑洞正在畸变"。
**为什么**：给用户"我在驱动这个时空场"的核心反馈，区别于死的进度条。
**验收**：收缩与手指实时同步（≤2 帧），半径随进度收 30–55%。

## 铁律 2 ｜ 因果雷达硬显
**法则**：1px 线化身游标卡尺；游标扫过某条内耗文案的锚点，那条文案**"啪"地满亮**，未扫到的**完全不绘制**。进/出带迟滞防闪。
**为什么（易理证明）**：爻是二进制，没有半亮的爻。渐隐本身背叛了爻的本性——**我们不淡入淡出，因为爻不淡入淡出。**
**验收**：全程搜不到该文案的中间灰度。

## 铁律 3 ｜ 物理爻变（沙化垂直下坠）
**法则**：用户**反本能拉到极限**、把线狠狠拉断的瞬间（物极必反/老阳变阴），被命中过的**心魔文案本体**碎成粒子、统一逐帧自由落体（`vy = 2.0 + rand·3.5`）垂直坠落；线头 `Math.sin` 高频衰减回弹；急停 + 脆响。
**为什么**：撕断的必须是"他的标签/心魔"本身，不是代理卡片。看见标签→亲手破除它=破标签化心魔的字面实现。
**验收**：碎的是被命中的字；坠落物理全链统一；触发=主动撕，非滑满自动。

# 第四部 · 手感宪法（物理常量真源）

> 细则见《黄金样板屏物理常量标尺规格》。此处只立纲。

- **A 响应地基**：≤2 帧跟手、死区、拖拽增益（0.5×屏宽=满程）、松手棘轮回弹。
- **B 弓弦张力**：线随拉拽弯成神经弓弦；阻力随张力升；临界预紧；崩断 sin 回弹。
- **C 雷达硬显**：见铁律 2。
- **D 向心收缩**：见铁律 1。
- **E 档位锁止（采样）+ 三通道反馈**：过档同帧触发 声 + 震 + 准星 180° 翻面。
- **F 自演示 affordance（可发现性=Boss）**：静息时线/光点自己示范"往哪拉"；首次推进后停；**零说明文字**。
- **G 沙化/爻变**：见铁律 3。
- **H 触感降级矩阵**：`vibrate` 有则用；**iOS 网页无振动**→ 声效 + 屏震 + 急停顶替。**声音是被严重低估的最高 ROI 通道**：一声金属脆响 + 一帧急停，胜过十层粒子。
- **I 性能**：固定步长物理 / DPR≤2 / 粒子池复用 / ≥55fps。
- **J 可逆退出**：无按钮 ≠ 困在舱里；提供不破坏隐喻的物理回退（如向下沉降）。

> 撕断的"断不断"必须是**一条可发现的确定规律**（拖到临界→线锁死发烫+全屏渐强颤动→继续拉满即断），绝不能靠手速、绝不能时灵时不灵。

# 第八部 · 商业飞轮的伦理底盘

> 动态人格产品的变现，必须与其哲学自洽，否则反噬。

**铁律：解脱免费，永久收费。**
- **情绪释放永远先白给**：那一撕、那一崩、看见自己的轨迹——免费。
- **付费只锁"演化"**：轨迹的**长期留存、定期复测、演化对比、更深层爻象**。
- **为什么这是对的**：动态人格 = 值得长期追踪 = 订阅式"看自己怎么变"，**诚实且自洽**；而在用户被撕开、最脆弱的情绪峰值弹付费墙拦住解脱，会被读成精神操控，招致退款与信任崩塌。
- 我们卖的是"想留住、想再来、想看自己变成什么样"，不是"想解脱却被拦"。
